# Utilisation de l'Intelligence Artificielle dans mon projet

## Introduction
Dans le cadre de mon projet, j'ai utilisé l'outil d'intelligence artificielle ChatGPT. Je l'ai utilisé afin de pouvoir analyser et comparer une autre approche sur quelques bouts de codes.

## Utilisation de ChatGPT
J'ai utilisé ChatGPT pour comparer notre code de l'app Android au niveau de sa barre de recherche avec un code proposé par ce modèle. J'ai ensuite effectuer des modification sur le notre déjà existant pour simplifier, optimiser et rendre plus lisible le code de notre application. 

## Conclusion
L'utilisation à bon escient de ChatGPT a permis d'avoir un comparatif pour mon code et un regard différent sur mon travail au sein du projet. Cela m'a donnée une plus grande ouverture sur les différentes possibilités qui était applicables dans le cadre de l'application. Cette intégration m'a été d'une bonne aide.
