# Utilisation de l'Intelligence Artificielle dans mon projet

## Introduction
Dans le cadre de mon projet, j'ai utilisé l'outil d'intelligence artificielle Gemini. J'ai utilisé cette IA principalement pour découvrir de nouvelles approches techniques et découvrir de nouvelles bibliothèques.

## Utilisation de Gemini
J'ai utilisé Gemini pour créer un premier jet de la communication entre l'application Android et le site web afin de récupérer les différentes informations concernant les failles pour ensuite les afficher dans nos différentes activités. En utilisant ce premier jet comme modèle pour gérer la communication entre l'application Android et le site web, j'ai pu découvrir la bibliothèque Volley et j'ai donc découvert comment l'utiliser pour gérer la communication API. J'ai comparé/améliorer le code fourni par Gemini via différentes sources sur Internet, afin de m'approprier correctement ces nouvelles connaissances.

## Conclusion
L'intégration de Gemini a été très bénéfique pour le bon déroulement de mon projet. Elle m'a permis de combiner la réussite du projet et l'apprentissage personnel, en me permettant d'en apprendre plus sur la communication API entre l'application Android et le site web via l'utilisation de la bibliothèque Volley.
