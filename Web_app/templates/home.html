Bonjour et bienvenue sur notre site !

Notre projet consiste à développer une application Java capable de scanner un réseau privé à la recherche de vulnérabilités. L'interface web vous permet de visualiser les résultats des scans, stockés dans notre base de données, et de suivre l'état de chaque analyse.

Une application Android est également disponible pour requêter l'API et obtenir en temps réel les données des scans effectués par notre outil Java.
